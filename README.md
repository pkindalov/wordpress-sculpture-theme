# Sculpture Theme

Professional WordPress child theme for Astra, designed for sculptor portfolios.

## Requirements

- WordPress 5.0+
- Astra Theme (free version)
- PHP 7.4+

## Recommended Plugins

- Advanced Custom Fields (ACF)
- Custom Post Type UI
- Kadence Blocks

## Installation

1. Ensure Astra theme is installed and activated
2. Upload `sculpture-theme` folder to `/wp-content/themes/`
3. Activate Sculpture Theme from Appearance → Themes

## Features

- ✅ Modular code structure
- ✅ ACF shortcode support
- ✅ Custom post type templates
- ✅ Responsive design
- ✅ Performance optimized

## File Structure
```
sculpture-theme/
├── inc/                 - PHP modules
├── assets/             - CSS, JS, Images
├── templates/          - Page templates
└── template-parts/     - Reusable components
```
